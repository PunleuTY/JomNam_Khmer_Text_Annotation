# Khmer Data Annotation Tool

![License Badge](https://img.shields.io/badge/license-MIT-green)
![Version Badge](https://img.shields.io/badge/version-1.0-blue)

A web-based tool designed to simplify and speed up the annotation of Khmer text, enabling efficient preparation of datasets for AI and machine learning projects using Khmer script.

---

## 🚀 Features
- User-friendly web interface
- Supports text annotation, labeling, and exporting (JSON, YOLO)

---

## 🛠️ Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: GO
- Database: MongoDB
- ML Engineer: Python, Tesseract, FastAPI

---

##  💾 Setup & Testing
1. Clone the repo: 
```bash 
git clone https://github.com/PunleuTY/Khmer-Data-Annotation-Project.git
cd Khmer-Data-Annotation-Project
```
2. Backend
```bash
cd backend
go run server.go
```
3. Frontend
```bash
cd frontend
npm install
npm run dev
```
4. ML
```bash
cd ML_V3_Final
python main_server.py
```
